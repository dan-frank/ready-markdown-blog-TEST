Features

  • [x] more than 150 supported languages (standard version)
  • [x] 30 supported languages (libre version)
  • [x] Material Design 3 (You)
  • [x] Dark and light theme
  • [x] Entirely Open Source
  • [X] Written in Jetpack Compose
  • [X] Support native Android sharing system
  • [X] Multilingual interface

<!-- ---------- App Versions ---------- -->

Versions

There are currently two different versions of TY available:

Translate You Libre

Only supports the <a href="https://github.com/LibreTranslate/LibreTranslate">LibreTranslate API</a> and therefore is fully libre.

Translate You

NonFreeNet anti-feature: Due to supporting translation engines that are not open source.

Additionally supports:

  • <a href="https://github.com/thedaviddelta/lingva-translate">LingvaTranslate</a> which uses Google Translate® to fetch translations.
  • <a href="https://www.deepl.com/translator">Deepl translator®</a>.
  • <a href="https://mymemory.translated.net/">MyMemory translator®</a>.

<!-- ---------- Contribution ---------- -->

Contribution

All contributions are welcome!
Feel free to suggest new features or new APIs that could be supported.
Inform us of a bug by opening a bug report.
If you are a developer and wish to contribute to the app, please fork the project and submit a pull request.
